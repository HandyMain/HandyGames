# HandyApp - Ultimate Kids Learning & Adventure Suite

A comprehensive Progressive Web App (PWA) designed to combine education, logic puzzles, augmented reality (AR), and creative play into a single "Super App" for children. Powered by React, Tailwind CSS, and Google Gemini AI.

    *   *New Jobs:* Car Engine Tune-up, Garden Planter Box, Leaky Pipe, Wobbly Shelf.
    *   *Mechanics:* Interactive tools (Wrench, Drill, Oil Can, Hammer) with haptic feedback and physics.
    *   *Optimization:* Fully responsive layout for mobile devices.
*   **Happy Farmer:** A tycoon-style farming sim.
    *   Inventory system (Silo), Market selling, Animal husbandry (Barn), and Tech upgrades (Sprinklers).
*   **Neon Blade AR:** A high-fidelity AR slicing game using the camera and custom 3D projection.
*   **Mobile Optimization:**
    *   Hidden navigation bars during gameplay for immersive full-screen experiences.
    *   Dynamic sizing for various screen widths (phones vs tablets).
